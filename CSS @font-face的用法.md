# CSS @font-face的用法

```css
@font-face {
  font-family: "阿里妈妈刀隶体";
  src: url("//at.alicdn.com/wf/webfont/*/*.woff");
}
```

```html
<span style="font-family:'阿里妈妈刀隶体';">巷子里的饰品小铺</span>
```
